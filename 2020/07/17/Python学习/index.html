程序二，成绩判断

> core = int(input("请输入成绩"));  //因为输入的字符属于字符串，所以需要使用int()，将输入的字符串转换为int类型，否则报错。
> if core < 60:
>     print("成绩不合格");
> elif core >= 60 and core < 70:  //这里elif代表的就是java中的else if 分支判断。
>     print("成绩合格");
> elif core >= 70 and core < 80:
>     print("成绩中等");
> elif core >= 80 and core < 90:
>     print("成绩上等");
> elif core >= 90 and core < 100:
>     print("成绩优秀");
> else:
>     print("成绩输入错误");

程序三：字符串赋值

> string = "python";     //每个字符串中的字符对应一个索引，从0到5
> print(string[0:2]);    //取索引开始为0，个数为2的字符，即“py”，
> print(string[0:1000]); //结果为python，与String[0:6]结果相同
> print(string[0:1000:2]);//结果为“pto”，从p开始，每次中间隔两个进行取值（步长为2）。
> print(string[-1]);      //结果为‘n’，字符串可以倒着进行索引编码。从-1开始。
> print(string[-1:-1000:-1]);//结果为nohtyp ，反向取值。
> if "py" in string:      //结果为"正确",表示string这个字符串中含有py两个字符
>     print("正确");
